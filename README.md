# GestionGym

Proyecto de gestión de membresías para gimnasio.

## Ejecutar el backend de consola

```bash
python main.py
```

## Ejecutar el frontend PyQt5

```bash
python frontend.py
```

## Ejecutar el frontend web

```bash
python web_frontend.py
```

Abrir luego en el navegador:

```bash
http://127.0.0.1:5000
```

## Dependencias
```bash
pip install -r requirements.txt
```

El frontend utiliza `PyQt5` para mostrar:
- clientes
- planes
- suscripciones
- registro de auditoría y alertas
- operaciones de asignación, renovación, suspensión y cancelación
